# 📅 Sistema de Eventos - Django

Projeto web desenvolvido com Django para gerenciamento de eventos.

## 🚀 Funcionalidades

- Criar eventos
- Listar eventos
- Visualizar detalhes de um evento
- Controle de quantidade de participantes
- Templates personalizados

## 🛠 Tecnologias utilizadas

- Python 3
- Django
- SQLite
- HTML
